Mongo 数据库管理入口
说明：提供开发环境（Workspace）和部署环境（Deploy）的 MongoDB 数据库管理功能。
开发环境需要 JWT 鉴权（userId + appId），部署环境需要 JWT 鉴权（appId）。
建议用 URL hash 传 token：#token=Bearer%20...（不会出现在 access log）。

开发环境数据库管理 可用

管理用户开发容器的 MongoDB 数据库（Workspace Mongo）。支持查询、插入、更新、删除等操作。

进入开发环境数据库

部署环境数据库管理 可用

管理应用部署运行时的 MongoDB 数据库（Deploy Mongo）。支持查询、插入、更新、删除等操作。
