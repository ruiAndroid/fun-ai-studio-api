Mongo Explorer 入口
说明：Mongo Explorer 提供开发态和部署态的 MongoDB 数据管理功能。
建议用 URL hash 传 token：#token=Bearer%20eyJ...（不会出现在 access log）。

Workspace Mongo Explorer 开发态

管理开发环境的 MongoDB 数据（需要 userId + appId + JWT）。支持查询、编辑、删除、新增记录和创建集合。

进入 Workspace Mongo

Deploy Mongo Explorer 部署态

管理部署环境的 MongoDB 数据（需要 appId + JWT）。支持查询、编辑、删除、新增记录和创建集合。

进入 Deploy Mongo
